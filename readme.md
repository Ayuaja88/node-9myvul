### How to Use this project
#### Backend
1. Clone this repository
2. Go to folder backend
6. Install dependency with command : 
```javascript
npm install
```
7. Run application backend with command: 
```javascript
npm start
```



#### Frontend
1. Go to frontend
2. Install dependency : 
```javascript
npm install
```
3. Run your application frontend : 
```javascript
npm start
```
